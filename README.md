This is a small list of my small scale personal projects:
- AirBnB price prediction model: A data science/ML project where I trained and tuned an XGBoost regression model on 48,000+ NYC Airbnb listings to predict nightly prices and then validated against an independent out-of-distribution dataset to test real-world generalisation
- Homeboy-Smart Apartment: An IoT project using Arduino and simple sensors/motors to simulate a smart apartment with a centralized control system. Built as a team, I was the project lead and handled all the code
- Website Redesign: A small website redesign of a small local restaurant for a first-year unit. Tackled customer research and UI design.
